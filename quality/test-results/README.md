# Prompt And Planning Test Results

Use this directory to record dated test runs before any prompt packet or scene-plan template is treated as production-ready.

Recommended artifacts:

- dated Markdown summaries
- side-by-side comparisons
- score tables for continuity, composition, timing, and style fidelity
- notes about which local model stack or Blender setup was used

If a plan or prompt fails, record why. Memory is not a pipeline.
