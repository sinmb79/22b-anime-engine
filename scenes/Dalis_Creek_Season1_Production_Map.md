## 제작 큐 제안

### 큐 A: 시리즈 기반 구축

- EP01을 기준 에피소드로 scene JSON 분해
- 주요 로케이션과 캐릭터 canonical asset ID 확정
- previz와 Blender 최종 렌더의 타이밍 일치 검증

### 큐 B: 시즌 확장

- EP02~EP05까지 계절/가족/액션 축 검증
- 재사용 가능한 배경과 소품 묶음 확정

### 큐 C: 쇼츠 파생

- 각 에피소드의 `⭐ 숏폼 추출 구간`에서 20~60초 클립 재구성
- 플랫폼별 메타데이터 자동화

## 메모

쇼츠가 앞서면 감정선이 잘립니다.
롱폼이 먼저 서야, 쇼츠도 기억에 남습니다.
